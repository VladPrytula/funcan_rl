What's a "heuristic"?

Since the AI can't search billions of positions, it estimates position value by counting potential winning lines, threats, and control.

Think of it like chess intuition - you evaluate positions without calculating every possible game to the end.

Theory Connection (Week 24)

This AI uses minimax search, which is equivalent to value iteration on a game tree. The algorithm computes the optimal value function V*(s) via backward induction.
